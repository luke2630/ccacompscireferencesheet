Introduction to HTML

A HTML page must include a <!DOCTYPE html> tag to let the browser know that this page will be in html. After that it is important to include a <html> tag at the beginning and a </html> tag at the end of your code. A html code must also include a head and body tag which will include the contents of the page. There are many different kinds of html tags that can do a variety of things. Such as the headings tags which create headings or the break tag that can create spaces between blocks of texts.

Headings

Description

The <h1> tag and all other h tags are used to create headings within a browser. The number that comes after the h determines how important that heading is to the browser.

Example

    <!DOCTYPE html>
      <html>
        <head>
        </head>
        <body>
          <h2>MAIN IDEA</h2>
            <h4>POINT 1</h4>
            <h4>POINT 2</h4>
            <h4>POINT 3</h4>
        </body>
      </html>

Browser Display

MAIN IDEA

POINT 1

POINT 2

POINT 3

Paragraphs

Description

The <p> is where the majority of content that is in a website is located. It contains all of the information that the author wants to convey that isn't stated in the headings.

Example

<p>This is a paragraph, it contains information.</p>

Browser Display

This is a paragraph, it contains information.

Breaks

Description

The <br> are used to create lines between different paragraphs, links, headings etc. They simple create a space between the two different lines of code. Similar to hitting enter and creating a space between two paragraphs.

Example

<h4>Heading 1</h4>
<br>
<p>This is the paragraph that relates to the heading</p>

Browser Display

Heading 1


This is the paragraph that relates to the heading

Quotation

Description

The <q> tag is used to add quotations to a paragraph or heading. It is merely inserted around the word(s) that need to be placed within quotations.

Example

<p>This paragraph contains some of the <q>best</q> literature that the world has to offer</p>

Browser Display

This paragraph contains some of the best literature that the world has to offer

Lines

Description

Example

<hr>

Comments

Description

<!-- --> tags are used to create small notes or comments within your code that won't affect the program itself. It merely helps the person reading the code to better understand what the code is doing and how it does it.

Example

<!-- This is the heading and it displays the large word "Heading" -->
<h4>Heading</h4>

Links

Description

The <a href=""> </a> tags are used to insert links into your website. These tags allow you to create a small portion of text and make it into a link which can direct them to other parts of your website or an entirely different website.

Example

<a href="https://www.google.com/">Link to Google Home Page</a>

Lists

Description

There are two types of lists, ordered(<ol>) and unordered(<ul>). After you have created the original list, you can then add information into the list using the <li>. This allows you to just add lists to your webpage.

Example

<!--unordered list>
<ul>
  <li>List Item 1&lot/li>
  <li>List Item 2&lot/li>
  <li>List Item 3&lot/li>
</ul>
<!--ordered list>
<ol>
  <li>List Item 1&lot/li>
  <li>List Item 2&lot/li>
  <li>List Item 3&lot/li>
</ol>

Tables

Description

Tables allow a web developer to create sets of information organized into rows and columns. To create one, first create a <table> tag to be able to create the table. Then add a row using the <tr> tag and to make that row a header, add pieces of information using the <th> tags and to just add another row of information use the <td> tag.

Example

<table>
  <tr>
    <th>First Name</th>
    <Last Name</th>
  </tr>
  <tr>
    <td>Bob</td>
    <td>Ross</td>
  </tr>
  <tr>
    <td>Joe</td>
    <td>Smith</td>
  </tr>
  <tr>
    <td>Jeff</td>
    <td>White</td>
  </tr>
</table>

Joe Smith
Jeff White

Description

Example

  <p>Paragraph</p>
